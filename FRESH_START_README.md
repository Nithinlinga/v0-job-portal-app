## 🎬 Quick Start

**If you're in a hurry:**

1. Create a new Supabase project
2. Run the SQL scripts from `SQL_SETUP_REFERENCE.md` in Supabase
3. Get your credentials (URL and Anon Key) from Supabase Settings
4. Create a new v0 project with a GitHub repo
5. Copy all code files from this project
6. Add environment variables with your Supabase credentials
7. Done! 🎉

**Total time: ~30 minutes**

## 💡 Pro Tips

1. **Save your Supabase password** - You'll need it later
2. **Copy credentials carefully** - No spaces or typos
3. **Restart dev server** - After adding environment variables
4. **Test with two accounts** - One HR, one Student (to see it working)
5. **Check the console** - If something breaks, browser console shows errors
